"# Ultimate-Arcade-Opengl-game" 
Ultimate Arcade is a multi-game interactive application developed using C++ and OpenGL/GLUT. The application presents three distinct classic arcade games: Snake, Car Racing, and Space Shooter, all within a single unified executable. Players are presented with an animated main menu from which they can select any game, and they may return to the menu at any time using the ESC key. Each game maintains its own score, lives system, and difficulty progression, ensuring a unique and escalating challenge for the player.
The Snake game challenges the player to navigate a growing snake across a 30×20 grid, consuming food pellets while avoiding collisions with walls or the snake's own body. The Car Racing game places the player in control of a vehicle on a three-lane road, requiring lane changes to avoid oncoming enemy cars while accumulating score. The Space Shooter game is inspired by classic Space Invaders, where the player controls a spaceship to destroy waves of alien enemies before they reach the bottom of the screen.
